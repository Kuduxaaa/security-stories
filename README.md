# security-stories

I keep running into people who want to hack me. So I started writing about it.

This is my collection of real security incidents — things that actually happened to me, fully analyzed and documented. Technical enough to be useful, sarcastic enough to be entertaining.

If someone's going to waste my time trying to compromise my machine, the least I can do is turn it into content.

## Stories
